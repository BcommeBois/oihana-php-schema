TokenRequestField uses ConstantsTrait

TokenRequestField.php : 18

Form field names sent in the `application/x-www-form-urlencoded` body of a request to the OAuth 2.0 token endpoint.

Covers RFC 6749 §4.4 (client_credentials grant) and RFC 7521 (client_assertion extension for asymmetric client authentication).
