## 📐 How this wiki is organised

- **`en/`** — English-language guides.
- **`fr/`** — Guides en français.
- Each language folder mirrors the same structure: an index (`README.md`) and one Markdown file per topic.
- Topics are written from scratch and target humans — they complement (and never duplicate) the API reference auto-generated from PHPDoc comments.

## 🛠️ Auto-generated API reference

The full class-by-class API documentation is generated by phpDocumentor and lives in [`docs/`](../docs). Regenerate it with:

```bash
composer doc
```

## ✍️ Contributing a page

1. Add the English version under `wiki/en/`.
2. Add the French translation under `wiki/fr/` (use the same filename in French where it helps readers — for example `getting-started.md` becomes `demarrage.md`).
3. Update both `wiki/en/README.md` and `wiki/fr/README.md` so the new page is discoverable.
4. Open a pull request — both languages should land in the same commit.
